Бизнес-функция: Онлайн-регистрация на рейс

1. ГЛОССАРИЙ (чтобы все понимали о чем речь)
• PNR - номер брони в системе авиакомпании
• Бортовой талон - электронный посадочный
• Онлайн-регистрация - получение посадочного без стоек в аэропорту
• Срок регистрации - можно зарегиться за 24-2 часа до вылета

2. ЦЕЛИ (зачем это вообще нужно)
• Убрать очереди в аэропорту
• Чтобы 60% пассажиров регились онлайн
• Разгрузить сотрудников в аэропорту
• Сделать удобно для людей

3. КАК РАБОТАЕТ (процесс по шагам)
1. Человек вводит номер брони и фамилию
2. Система проверяет можно ли зарегиться
3. Человек выбирает место (если есть)
4. Человек получает посадочный на почту/телефон

4. ЧЕ ХОЧУТ ЛЮДИ (user story)
• "Хочу зарегиться онлайн чтобы не стоять в очередях"
• "Хочу выбрать место у окна для фото"
• "Хочу посадочный на телефон чтобы не печатать"

5. ОГРАНИЧЕНИЯ (что нельзя)
• Нельзя зарегиться раньше чем за 24 часа
• Нельзя если больше 6 человек в броне
• Нельзя выбрать места у аварийных выходов
• Нельзя для чартерных рейсов

6. КОГДА МОЖНО НАЧИНАТЬ ДЕЛАТЬ (DoR)
✓ Дизайнер нарисовал как это выглядит
✓ Есть доступ к системе авиакомпании
✓ Юристы проверили все документы
✓ Сделали тестовые номера броней

7. КОГДА СЧИТАТЬ ГОТОВЫМ (DoD)
✓ Регистрация работает на тестовом сервере
✓ Письма с посадочными приходят на почту
✓ Протестировали все возможные ошибки
✓ QR-коды работают правильно
✓ Написали инструкцию для поддержки
