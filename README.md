Бизнес-функция: Онлайн-регистрация на рейс

Глоссарий

PNR - номер бронирования в системе авиакомпании

Бортовой талон - электронный посадочный талон

Электронная регистрация - процесс получения посадочного талона без обращения в стойку регистрации

Срок регистрации - временное окно для прохождения онлайн-регистрации (обычно за 24-2 часа до вылета)

Бизнес-цели

Сократить очередь в аэропорту на 40%

Увеличить процент прошедших онлайн-регистрацию с 20% до 60%

Снизить нагрузку на staff в аэропорту

Улучшить customer experience

Бизнес-процесс

Пользователь вводит PNR и фамилию

Система проверяет возможность онлайн-регистрации (сроки, маршрут, тип билета)

Пользователь выбирает места (если доступно)

Пользователь указывает контактные данные для уведомлений

Система генерирует посадочный талон

Пользователь получает бортовой талон на email/в приложение

User Story

Как пассажир, я хочу пройти регистрацию онлайн, чтобы не стоять в очереди в аэропорту

Как пассажир, я хочу выбрать место у окна, чтобы смотреть вид во время полета

Как пассажир, я хочу получить посадочный на телефон, чтобы не печатать его

Ограничения

Нельзя пройти регистрацию раньше чем за 24 часа до вылета

Нельзя пройти регистрацию если в бронировании больше 6 пассажиров

Нельзя выбрать места экстренного выхода

Не доступно для рейсов с особыми условиями (чиартер, визы)

Критерии готовности (DoR)

Дизайн макеты готовы

API авиасистемы доступно

Юридические требования к данным пассажиров согласованы

Тестовые PNR созданы

Критерии завершённости (DoD)

Регистрация работает в тестовом окружении

Интеграция с email-рассылкой настроена

Протестированы все сценарии ошибок
